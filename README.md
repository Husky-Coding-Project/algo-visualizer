# algo-visualizer

The Express server should be started before running the React client.

## **Running guide**:

If you don't have Node.js, install it from the [official website](https://nodejs.org/en/).

### **Start ExpressJS server**

From main directory:

1. Change directory to the `server` folder.
2. Install necessary packages.
3. Start the server.

```bash
cd server
npm install
npm start
```

### **Start React client**

From main directory:

1. Change directory to the `client` folder.
2. Install necessary packages.
3. Start the server.

```bash
cd client
npm install
npm start
```
